# AURUM Fine Dining Website

A modern, luxury restaurant website built with HTML, CSS, and JavaScript.

## Features

- Elegant hero section with slideshow
- Responsive navigation menu
- Animated preloader
- Smooth scroll reveal effects
- Menu tabs for starters, mains, desserts, and drinks
- Gallery with lightbox viewer
- Chef showcase section
- Testimonials slider
- Reservation form with validation-style feedback
- Custom cursor and premium UI design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
- Unsplash images

## Folder Structure

```bash
restaurant-website/
├── index.html
├── css/
│   └── style.css
└── js/
    └── main.js
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a browser.

For the best experience, use VS Code Live Server.

## Notes

- The website uses online images and fonts, so an internet connection is recommended.
- This project is a front-end demo and does not send reservation data to a real backend yet.
